# AirFare Radar

AirFare Radar is a Vue 3 + TypeScript prototype for discovering standout flight deals with a
dashboard, map placeholder, and blind-box experience scaffold.

## Stack

- Vue 3
- Vite
- TypeScript
- Pinia
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Included in Phase 1

- Base app shell with top navigation
- Responsive container layout
- `FlightTicket` TypeScript interface
- 10 pieces of high-quality mock fare data
- Pinia store for shared fare state
